Convert data for a seglearn pipeline¶

First, get measurement data

>>> from datatools import MeasurementDataReader, MeasurementSeries, to_ts_data
>>> mdr = MeasurementDataReader()
>>> data_dict = mdr.query().filter_by(MeasurementSeries == "pythagoras-01-20190705").get()

Let’s prepare the data_dict for seglearn’s Pype using TS_Data

>>> Xt, Xc, y = to_ts_data(data_dict)
>>> from seglearn.base import TS_Data
>>> X = TS_Data(Xt, Xc)
